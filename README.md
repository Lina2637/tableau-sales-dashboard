# Tableau Sales Dashboard

## Overview
This project is a beginner-friendly Tableau dashboard created to analyze sales performance across product categories and regions using the Superstore dataset.

The project demonstrates basic business intelligence and data visualization skills using Tableau Public.

---

## Dashboard Features
- Sales analysis by product category
- Regional sales comparison
- Stacked bar chart visualization
- Interactive region filter
- Clean and minimal dashboard design
- Sorted category performance view

---

## Tools Used
- Tableau Public
- Superstore Dataset
- Data Visualization
- Business Intelligence Concepts

---

## Key Insights
- Technology products generated the highest overall sales.
- Sales performance varies across different regions.
- Stacked visualizations help compare regional contributions within categories.
